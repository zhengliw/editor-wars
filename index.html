Schreib.

Um verschiedene Arten von Text und Code schreiben und bearbeiten zu können, braucht man einen Texteditor. Und dafür gibt es viele Möglichkeiten.

Diese Seite hilft dir dabei, einen Texteditor auszuwählen.

Was einen Editor ausmacht (und was nicht)

Nicht alle Programme, die Textdateien öffnen und bearbeiten können, lassen sich als einen Texteditor bezeichnen. Man unterscheidet zwischen Texteditoren (Vim, Code...) und Textverarbeitungsprogrammen (Word, Writer...). Texteditoren arbeiten mit reinem Text (plain text) - die Dateien enthalten nur Text oder Code und können mit jedem anderen Texteditor geöffnet werden. Man sieht alles, was in der Datei enthalten ist.

Textverarbeitungsprogramme speichern zwar auch das Geschriebene in Dateien, jedoch in eigenen Dateiformaten, wo sie neben Text noch „unsichbare“ Daten wie Layoutparameter oder Dokumenteinstellungen hinterlegen. Sie eignen sich dadurch besser für Dokumente, die ein bestimmtes Layout oder eingebettete Elemente wie Bilder benötigen.
